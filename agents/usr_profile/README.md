# usr_profile Agent

用户画像提取专家，用于从对话历史中提取用户的详细画像信息。

## 功能

该 agent 专门用于分析对话内容，提取用户的十个维度画像：

1. **对...感兴趣** - 表达出兴趣，但不一定了解
2. **喜欢（偏好）** - 有兴趣，有明确偏好  
3. **特殊爱好** - 有兴趣，有明确偏好，但这个爱好本身比较小众/特别/私密
4. **沉迷** - 深度沉迷
5. **性癖** - 对话、亲密行为中表现出来的强势、弱势、BDSM倾向、智性恋等细化标签
6. **擅长领域** - 用户自身擅长的技能，如健身、厨艺、二次元等细化标签
7. **性格** - 用户自身的性格特点，如孤僻、外向、强迫症等细化标签
8. **对{role}的承诺** - 用户对角色做出的承诺
9. **对{role}的期望** - 用户对角色的期望，如无条件的爱/称呼自己为太太等
10. **愿望** - 用户希望未来的某件事发生

## 使用方法

输入对话历史数据，agent 会按照固定格式输出十个维度的用户画像信息。

## 输出格式

```
<对...感兴趣>美食、风景
<喜欢（偏好）>不吃辣、汤汤水水的食物、晴天
<特殊爱好>
<沉迷>角色扮演
<性癖>追求刺激、强势
<擅长领域>
<性格>主动、幽默、敏感
<对{role}的承诺>有时间多陪陪{role}
<对{role}的期望（希望{role}能...）>把{role}当猫养
<愿望（希望...发生）>与{role}结婚
```
